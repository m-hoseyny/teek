# Top 10 Free Subtitle Font Pack

This pack is curated for subtitle/caption readability and includes only fonts
from Google Fonts that are free for commercial use (mostly SIL OFL 1.1).

Install command:

```bash
./backend/bin/install_subtitle_font_pack.sh
```

Installed files:

- `Anton-Regular.ttf`
- `ArchivoBlack-Regular.ttf`
- `BarlowCondensed-SemiBold.ttf`
- `BebasNeue-Regular.ttf`
- `Inter-Variable.ttf`
- `Lato-Bold.ttf`
- `NotoSans-Regular.ttf` (Universal UTF-8 support)
- `OpenSans-Variable.ttf`
- `Oswald-Variable.ttf`
- `Poppins-SemiBold.ttf`
- `Roboto-Variable.ttf`

Source and license references:

- Google Fonts FAQ (commercial use): https://developers.google.com/fonts/faq
- SIL Open Font License 1.1: https://openfontlicense.org/
- Google Fonts GitHub repository: https://github.com/google/fonts
